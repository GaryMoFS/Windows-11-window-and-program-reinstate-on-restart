# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project follows Semantic Versioning.

## [0.1.0] - 2026-02-14

### Added
- Initial public release
- Save/restore named window layout presets
- System tray menu and desktop context menu integration
- Preset manager (restore, rename, delete, shortcut)
- Startup restore capability
- 4-quadrant layout save mode

### Changed
- Safer restore behavior for session-managed apps
- Browser tab automation disabled by default
